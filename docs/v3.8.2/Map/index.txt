<p>Map&#39;s keys can be of any type, and use <code><a href="/docs/v3.8.2/is()">Immutable.is</a></code> to determine key

<p>Because <code><a href="/docs/v3.8.2/is()">Immutable.is</a></code> returns equality based on value semantics, and
