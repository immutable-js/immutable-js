# isIndexed()

True if `maybeIndexed` is a Collection.Indexed, or any of its subclasses.

<Signature code={`isIndexed(maybeIndexed: unknown): boolean;`} />

```js
import { isIndexed, Map, List, Stack, Set } from 'immutable';

isIndexed([]); // false
isIndexed({}); // false
isIndexed(Map()); // false
isIndexed(List()); // true
isIndexed(Stack()); // true
isIndexed(Set()); // false
```
