# isKeyed()

True if `maybeKeyed` is a Collection.Keyed, or any of its subclasses.

<Signature code={`isKeyed(maybeKeyed: unknown): boolean;`} />

```js
import { isKeyed, Map, List, Stack } from 'immutable';
isKeyed([]); // false
isKeyed({}); // false
isKeyed(Map()); // true
isKeyed(List()); // false
isKeyed(Stack()); // false
```
